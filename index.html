🌐 Host a Static Website with GitHub Pages

A step-by-step guide to publishing your first web page using GitHub

🛠 Objective

Learn how to deploy a basic HTML website using GitHub Pages for free.

📦 Tools Required

  • GitHub account
  • Basic HTML knowledge
  • Optional: Git installed on your computer

📋 Step-by-Step Guide

  1. Create an index.html file with your website content.
  2. Create a new repository on GitHub (e.g., my-static-site).
  3. Upload your index.html (and style.css if any) to the repository.
  4. Go to Settings > Pages in the repo.
  5. Choose the main branch and root folder, then save.
  6. GitHub will give you a live link like:
    https://your-username.github.io/my-static-site/
